# Birthday Time Capsule

A standalone birthday invitation website built with plain HTML, CSS, and JavaScript. It uses an unlock screen, memory capsule gallery, wish wall, and future-message interaction.

## Customize

Edit the `capsuleData` object in `script.js`:

- `birthdayName`
- `madeBy`
- `sealedDate`
- `heroMini`
- `heroTitle`
- `heroText`
- `capsules`
- `wishes`
- `futureNotes`

Replace the placeholder images in `assets/` with real birthday photos. The default layout includes 12 capsule images.
