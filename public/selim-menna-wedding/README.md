# Maged & Shahd — Wedding Invitation

Static HTML/CSS/JS version of the romantic wedding invitation site. No build step, no framework — just open `index.html`.

## Files
- `index.html` — markup
- `styles.css` — all styles & torn-paper SVG masks
- `script.js` — preloader, countdown, scroll reveal, playlist form
- `assets/` — photos

## Deploy to GitHub Pages
1. Create a new GitHub repo and upload all files (keep the folder structure).
2. Repo Settings → Pages → Source: `main` branch, root.
3. Visit the published URL.

## Local preview
Just double-click `index.html`, or run:
```bash
python3 -m http.server 8000
```
