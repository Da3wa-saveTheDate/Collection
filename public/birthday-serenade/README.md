# Birthday Serenade Invitation

A romantic standalone birthday invitation built with plain HTML, CSS, and JavaScript.

## Customize

Edit the `serenade` object in `script.js`:

- `name`
- `date`
- `place`
- `dress`
- `heroKicker`
- `heroTitle`
- `heroText`
- `letter`
- `hiddenLines`
- `wish`
- `bouquetMessage`
- `moments`

Replace the placeholder images in `assets/` with real romantic photos. The default memory film includes 10 images.
