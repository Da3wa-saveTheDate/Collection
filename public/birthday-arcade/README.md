# Birthday Arcade Invitation

A standalone neon arcade birthday invitation built with plain HTML, CSS, and JavaScript.

## Customize

Edit the `arcade` object in `script.js`:

- `playerName`
- `ageLevel`
- `date`
- `place`
- `heroLabel`
- `heroText`
- `giftText`
- `giftCode`
- `prizes`

Replace the placeholder prize images in `assets/` with real photos, rewards, or birthday coupons. The bonus round is fully local JavaScript and does not need any backend.
